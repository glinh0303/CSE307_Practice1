HTML Links

Hyperlinks Syntax: The HTML < a > tag defines a hyperlink. It has the following syntax:

< a href="url"> link text < /a>

Images in HTML

    2. < body background="file name" > < /body>

    3. < img src='link to folder" title="Note">
